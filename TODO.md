# Future Enhancements

## Companion App
- Develop a companion app for iOS and Android.
- The app will allow users to configure the hosted instances they want to connect to.

## Server Federation (Instance Pooling)
- Implement a feature for admins of hosted instances to "connect" them and build a pool of servers.
- Every instance must have a unique ID for exchanging data.
- Develop protocols and APIs for exchanging matchmaking, game, and user data across federated instances.
